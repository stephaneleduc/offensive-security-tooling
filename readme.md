# Offensive Security Tooling

Ce dépôt présente les outils utilisés dans le cadre de mes activités
personnelles en cybersécurité offensive, en complément des laboratoires
Offensive Security, Hack The Box et CTF.

Les outils sont classés selon trois catégories :
- Outils utilisés dans le cadre d’activités de cybersécurité offensive, principalement depuis un environnement Kali Linux,
- Outils tiers issus de dépôts publics,
- Outils et scripts développés personnellement.

Tous les outils sont utilisés exclusivement dans des environnements autorisés et contrôlés.
